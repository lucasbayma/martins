    2. **NAV-02: Mouse click on sidebar items activates instantly.**
       - Click a project name to expand/collapse it → expected: immediate visual response.
       - Click a workspace entry inside the current project → expected: pane updates on the next frame.
       - Click a workspace entry in a DIFFERENT project (forces cross-project switch) → expected: PTY view updates immediately, right-sidebar diff list may flash or populate 10-100ms later (that is the CORRECT new behavior — the diff list arriving late is Phase 4's tradeoff).
       - Click a tab in the tab strip → expected: tab content swaps on the next frame.
       - Click a file in the right-sidebar diff list → expected: either a diff tab opens OR the diff preview updates; new-tab creation has its own perceptible cost (spawning tmux) and that is outside NAV-02 scope.
       - If any pre-pane-update click triggers a visible pause → flag **NAV-02 FAIL**.

    3. **NAV-03: Workspace switching shows target PTY immediately.**
       - Set up two workspaces, each with a visibly distinct PTY state. E.g., in workspace A run `echo "AAAA"; sleep 9999` and in workspace B run `echo "BBBB"; sleep 9999`. Leave both sleeping with their outputs on screen.
       - Switch rapidly between the two workspaces (keyboard-arrow + Enter, or click the sidebar entries).
       - **Expected:** the PTY view shows the target workspace's content on the VERY NEXT FRAME. No blank frame. No "loading" flash. No transient black screen. The `AAAA` / `BBBB` text is continuously visible in one or the other — never absent.
       - If you see a blank frame, flash, or stutter between switches → flag **NAV-03 FAIL**.
       - Optional bonus: open a repo with 1000+ files (e.g., a node_modules-heavy project, or a Linux kernel checkout) and repeat the switch. The diff list on the right sidebar may populate visibly later than the PTY view swap — that is CORRECT and not a fail condition. Fail only if the PTY swap itself stutters.

    4. **NAV-04: Tab switching is instantaneous.**
       - In a workspace with 3+ tabs, press F1, F2, F3 (or the number keys 1, 2, 3 in Normal mode).
       - **Expected:** each switch is indistinguishable from instant. Previous tab's output replaces in a single frame.
       - Click between tab headers in the tab strip.
       - **Expected:** same — single-frame swap.
