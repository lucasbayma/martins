<objective>
Extract draw orchestration from `src/app.rs` into a new `src/ui/draw.rs` module. This is the lowest-risk extraction because draw code only reads `App` state — no mutation, no async, no borrow-checker friction beyond holding `&mut self` for `ListState` widgets.

Purpose: Reduce `src/app.rs` by ~175 lines and establish the "free function takes &mut App" refactor pattern that the rest of the phase will follow.
Output: New `src/ui/draw.rs` with `draw()`, `status_bar()`, `menu_bar()` free functions. `App::draw` becomes a one-liner calling `crate::ui::draw::draw(self, frame)`.
</objective>
